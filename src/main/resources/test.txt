Post : ADD EMPLOYEE

URL : http://localhost:8080/api/employee

Input :
{
        "name": "Alex",
        "role": "System Engineer",
        "isPermanent": true,
        "salary": 20000.00
}

Output :
Employee added successfully : 1

------------------------------------------------

Get : GET LIST OF EMPLOYEE
